# Custom stylesheets

In addition to the CSS stylesheets included with the [Bootstrap3](https://github.com/pkp/bootstrap3) and [Health Sciences](https://github.com/pkp/healthSciences) parent themes, and the custom child themes developed for CUL Journals, most journals also have a custom stylesheet that is uploaded via the Admin. These stylesheets allow for minor style modifications that are specific to the journal (usually the journal editor's preferences regarding logo size, fonts, color accents, etc.).

The table below lists **all** journals on the multisite, including those that do not have a custom stylesheet. "Slug" is the journal's path. "Theme" is the active theme: the shared `healthSciencesChild` / `bootstrap3Child` child themes, a parent theme on its own, or the `journals-resources` hub. "JIRA" links the original setup ticket where one was recorded.
